Bootstap Input Groups..!

1) The .input-group class is a container to enhance an input by adding an icon, text or a button in front or behind the input field as a "help text".

2) Use .input-group-prepend to add the help text in front of the input, and .input-group-append to add it behind the input.

3) At last, add the .input-group-text class to style the specified help text.
